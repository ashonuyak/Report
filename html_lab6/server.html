Лабораторної роботи №6

WEB-вузол Apache Tomcat встановлений в java-application і повністю контролюється ним.

Перший варіант підключення Apache Tomcat:

Вказати у системі збірки веб-стартер, у якому підключається Apache Tomcat.

Другий варіант підключення Apache Tomcat:

Заходимо на офіційний сайт Apache Tomcat і завантажуємо .exe файл.

Запускаємо цей файл.

Обираємо опцію HostManager та натискаємо кнопку "Следующая".

Задаємо ім'я користувача та пароль. Додатково можна змінити порт TCP.

Вказуємо шлях встановлення Java JDK.

Вказуємо шлях встановлення Apache Tomcat.

Закінчуємо встановлення.

Динамiчнi елементи:
  • Accordion
  • Datepicker
  • Tabs
