Лабораторної роботи №5

На лабораторній роботі №5 ми придбали практичні навички роботи маніпулювання інформаційним вмістом Web-документа засобами бiблiотеки JQuery та iнших кросс-браузерних бiблiотек, засоби налаштування та тестування сценарiiв JavaScript. Ознайомилися з теоретичними відомостями до лабораторної роботи №5 ( бібліотека jQuery UI). Додали до сайту 2-3 динамічних елементи. Додали до сайту плагін галереї UniteGallery. Створили за допомогою бібліотеки: фотогалерею з можливістю перегляду збільшених зображень, слайдер фотографій, слайдер відеоматеріалів. Додали до сайту (або до окремої сторінки) кнопки соціальних мереж: VKontakte, Facebook, Twitter, Instagram, Pinterest. Додали до сайту інформери. Додали до сайту один із модулів jQuery для мультимедіа, а саме covervid.

Динамiчнi елементи:
  • Accordion
  • Datepicker
  • Tabs
